# ST-Narrative-HUD
🎭 Narrative HUD для SillyTavern

**Narrative HUD** — это масштабная надстройка для SillyTavern, которая превращает обычную текстовую ролевую игру в полноценную RPG. Мод добавляет динамические трекеры, систему прокачки, инвентарь, интерактивный календарь, кодекс и инфоблоки, которые генерируются нейросетью прямо во время чата.

## 🔥 Ключевые возможности

### ✨  «Машина времени» для лора (Narrative Storage)
Забудьте о том, что статы существуют только «здесь и сейчас». Narrative HUD обладает уникальной системой умного хранилища!
Мод сохраняет сгенерированные данные (кастомные блоки) для каждого отдельного сообщения и даже для каждого свайпа.
Хотите узнать, что думал персонаж 50 сообщений назад? Просто нажмите на иконку 📋 (Планшет) рядом со старым сообщением в чате!
Мод мгновенно подгрузит исторические данные в боковую панель, и вы сможете перечитать старые комментарии, дневник или ваши кастомные штуки.
Встроенный редактор JSON (кнопка </> над сообщением) позволяет вручную править сохраненные статы для любого свайпа.

## ⚙️ Технологии "Под капотом"

### ⚡ Лайт-режим (Экономия токенов)
Уникальный алгоритм фонового парсинга. Вместо того чтобы заставлять ИИ писать тяжелый JSON в каждом ответе (тратя токены генерации), мод делает два запроса:
1. Основной ИИ пишет красивый художественный ответ (без лишних инструкций).
2. Мод незаметно делает второй, "тихий" запрос с минимальным контекстом для извлечения статов.
**Результат: экономия контекста и никаких сломанных ответов бота!**

### 🧮 Встроенный калькулятор токенов
Боитесь перегрузить контекст? Во вкладке "Промпты" есть счетчик, который в реальном времени анализирует все ваши включенные модули, лор (глазики) и квесты, показывая точный расход токенов на системные инструкции.

### 🧠 Умная очистка (Менеджер базы данных)
Narrative HUD сохраняет лор для каждого отдельного свайпа (реролла) ИИ! Чтобы база не раздувалась, встроен инструмент "Умной очистки". Удаляйте "мертвые" свайпы или срезайте старую историю в один клик. Текущие статы при этом не пострадают.

## 🚀 Установка и Быстрый старт

1. Откройте SillyTavern.
2. Перейдите во вкладку **Extensions** (Расширения).
3. Нажмите **Install Extension** (Установить расширение).
4. Вставьте ссылку на этот репозиторий: `https://github.com/Gromova-Olga/ST-Narrative-HUD`
5. Перезагрузите SillyTavern (F5).

🎓 **Не знаете с чего начать?**
Откройте настройки мода (иконка ⚙️ на правой панели) -> перейдите в глобальные настройки (🎨 Внешний вид) -> вкладка «❓ FAQ & Обучение». 
Нажмите **«Начать интерактивный тур»** — скрипт сам проведет вас по всем меню, подсветит нужные кнопки и объяснит, как настроить ваш идеальный HUD!

## ❓ FAQ (Частые вопросы)

**В: Как работают инфоблоки (Мысли, Инвентарь врага, Погода)?**
О: Это кастомные поля. Вы можете создавать бесконечное количество таких кнопок в левом окне (Вкладка "Промпты"). Просто напишите инструкцию, и ИИ начнет генерировать для вас эти данные.

**В: Куда пропал выбор цвета у Трекеров?**
О: В глобальных настройках дизайна у вас включена галочка «Градиент полосок (от %%)». В этом режиме бары плавно меняют цвет автоматически (от зеленого к красному). Если хотите красить каждый трекер вручную — отключите её.

**В: Почему статы и отношения не обновляются после ответа бота?**
О: Убедитесь, что во вкладке "API" включен правильный профиль подключения. Если включен **Лайт-режим**, убедитесь, что API в принципе настроен и работает. Вы всегда можете принудительно обновить статы кнопкой [▶ Обновить данные] в меню правой панели.

**В: Я удалил чат в SillyTavern, но персонажи из него всё ещё висят в настройках мода!**
О: Зайдите во вкладку "База" -> "Умная очистка". В самом низу окна вы увидите список всех сохраненных папок-чатов. Найдите там удаленный чат и нажмите "✕ Удалить".
